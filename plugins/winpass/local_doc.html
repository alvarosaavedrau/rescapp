Reset Windows NT, 200x, XP, Vista, Seven and 10 password - Rescapp - Documentation
  • Step 1

    Rescapp searches for Windows System partitions on your drive.
  • Step 2

    If such partitions have been found, Rescapp displays a list of the partitions. At the same time you are prompted:

    Which Windows partition?

    You are supposed to choose your Windows partition, (you will probably only see one) and press the OK button.

  • Step 3

    Rescapp tries to backup your Windows registry files, on your Windows sytem.
  • Step 4

    You are optionally informed about the Windows registry files backup process.

    You will get (if an error happens):

    [ERROR] Backup of SAM directory went wrong.

  • Step 5

    Rescapp searches for System users based on previous information.
  • Step 6

    You are prompted:

    Which Windows user do you want to reset the password for?

    You are supposed to select the user whose password you want to reset.

  • Step 7

    You are informed about the Reset Windows NT, 200x, XP, Vista, Seven and 10 password process.

    You either get:

    [SUCCESS] Windows password was reset OK! :)

    or

    [ERROR] Windows password was not reset. Something went wrong! :(
