Change UEFI Boot order - Rescapp - Documentation

Newest computers start up thanks to the newest (from 2015 onwards) UEFI boot system.

Some of the UEFI firmware vendors might not allow you to change the different UEFI boot entries order. Or maybe you do not know how to access that option in your UEFI firmware UI. This option lets you change the UEFI boot entries order.

Please note that in some UEFI systems you might need to press an special hotkey (such as F8 or F10) in order to show up the UEFI boot menu in order to be able to choose your added UEFI boot entry.

  • Step 1

    Rescapp queries for your current UEFI Boot entries.
  • Step 2

    If such entries have been found, Rescapp displays them. At the same time you are prompted:

    Order UEFI boot entries in the order that your want.

    You are supposed to order those UEFI Boot entries and press the OK button.

  • Step 3

    You are informed about the Change UEFI Boot order process.

    You either get:

    [SUCCESS] The UEFI Boot order, was updated OK! :)

    or

    [ERROR] The UEFI Boot order, update went wrong! :(
