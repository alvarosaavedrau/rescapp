Change UEFI Boot order - Rescapp - Documentation
  • Step 1

    Rescapp queries for your current UEFI Boot entries.
  • Step 2

    If such entries have been found, Rescapp displays them. At the same time you are prompted:

    Order UEFI boot entries in the order that your want.

    You are supposed to order those UEFI Boot entries and press the OK button.

  • Step 3

    You are informed about the Change UEFI Boot order process.

    You either get:

    [SUCCESS] The UEFI Boot order, was updated OK! :)

    or

    [ERROR] The UEFI Boot order, update went wrong! :(
