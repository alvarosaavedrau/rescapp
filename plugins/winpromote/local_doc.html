Promote Windows NT, 200x, XP, Vista, Seven and 10 user to Administrator - Rescapp - Documentation
  • Step 1

    Rescapp searches for Windows system partitions, on your disk.
  • Step 2

    If such partitions have been found, Rescapp displays a list of the partitions. At the same time you are prompted:

    Which Windows partition?

    You are supposed to choose, your Windows partition, (you will probably only see one), and press the OK button.

  • Step 3

    Rescapp tries to backup your Windows registry files, on your Windows sytem.
  • Step 4

    You are optionally informed about the Windows registry files backup process.

    You will get (if an error happens):

    [ERROR] Backup of SAM directory went wrong.

  • Step 5

    Rescapp searches for System users, based on previous information.
  • Step 6

    You are prompted:

    Choose Windows user to promote to Admin

    You are supposed to select, the user which you want to, promote to Administrator.

  • Step 7

    You are informed about, the promote Windows NT, 200x, XP, Vista, Seven and 10 user to Administrator process.

    You either get:

    [SUCCESS] The user was promoted to Admin OK! :)

    or

    [ERROR] The user was not promoted to Admin. Something went wrong! :(
