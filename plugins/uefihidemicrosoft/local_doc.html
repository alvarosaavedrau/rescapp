Hide Microsoft Windows UEFI boot entry - Rescapp - Documentation
  • Step 1

    Rescapp searches for EFI partitions on your system.
  • Step 2

    If such partitions have been found, Rescapp displays them. At the same time you are prompted:

    Which EFI System partition?

    You are supposed to choose, your EFI System partition and press the OK button.

  • Step 3

    Rescapp searches for EFI files present, on your EFI System partition.
  • Step 4

    You are prompted:

    Select EFI file

    You are supposed to, choose the EFI file which you want to be the default boot entry.

  • Step 5

    You are informed about the Hide Microsoft Windows UEFI boot entry process.

    You either get:

    [SUCCESS] Hiding UEFI Microsoft Windows boot entry was OK ! :)

    or

    [ERROR] Hiding UEFI Microsoft Windows boot entry was not ok :(
