Share log - Rescapp - Documentation

Warning: This option requires a working Internet connection.

  • Step 1

    Rescapp searches for log files that have been generated by other Rescapp options.
  • Step 2

    You are shown a list, of the files found:

    Please choose a log file

    As you can see you choose between "_log.txt" and "_simple_log.txt" files. By default share "_log.txt" files.

  • Step 3

    Rescapp sends the log file contents, to a public pastebin service, (originally paste.debian.net) and gathers the associated url.
  • Step 4

    You are prompted:

    The url

    And also asked to: Right-click and select copy. You will be able to paste it later in the chat option

  • Step 5

    You are informed about the share log process.

    You either get:

    [SUCCESS] The log file was shared OK! :)

    or

    [ERROR] The log file sharing went wrong! :(

  • Step 6

    Optionally you can run the Chat option, (under the Support category in the main menu) and paste the copied url if you want help.

    But please be sure that someone is willing to help you before doing it.
