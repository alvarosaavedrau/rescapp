Create an hybrid MBR on a GPT partition - Rescapp - Documentation

This option lets you add an MBR to a GPT partition so that it gets converted into an hybrid MBR.



TIP: You probably do not need to use this option in 2020.

  • Step 1

    Rescapp searches for hard drives on your computer.
  • Step 2

    If such hard drives have been found, Rescapp displays a list of the hard drives. At the same time you are prompted:

    Which hard drive do you wish to create a hybrid MBR on?

    You are supposed to choose, the hard drive which has a GPT partition table, on which you want to have an hybrid MBR created.

  • Step 3

    Rescapp with all the gathered information, is going to create a hybrid MBR on your GPT hard drive.
  • Step 4

    You are informed about the hybrid mbr creation process.

    You either get:

    [SUCCESS] The hybrid MBR was created OK! :)

    or

    [ERROR] The hybrid MBR was not created. Something went wrong! :(
