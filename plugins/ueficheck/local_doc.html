Check if Rescatux has boot in UEFI mode - Rescapp - Documentation

There are two main ways of booting a computer nowadays. The old BIOS boot way and the newest (from 2015 onwards) UEFI boot way.

If your main system boots in UEFI boot mode you are advised to boot Rescatux in UEFI boot mode so that the UEFI boot recover options work ok.

If your main system boots in BIOS boot mode it doesn't matter too much in which boot mode you boot Rescatux.

This option shows you if you have booted in UEFI boot mode or if you have not booted in UEFI boot mode (that means you have booted in BIOS boot mode).

  • Step 1

    You are informed about the UEFI check process.

    You either get:

    [SUCCESS] This Live CD was booted in UEFI mode :)

    or

    [INFO] This Live CD was NOT booted in UEFI mode :|
