About Rescapp - Documentation

Check the README file in the source code, for finding all of the Rescapp components.

The purpose of Rescapp, is to provide the user with a graphical rescue tool that will assist them to regain access to a computer that has become non-bootable.

You will also be able to regain access to a Windows based system that has had the password forgotten.
