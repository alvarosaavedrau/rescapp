Reinstall Microsoft Windows UEFI boot entries - Rescapp - Documentation

Newest computers start up thanks to the newest (from 2015 onwards) UEFI boot system.

If you are playing with the UEFI boot entries you might end losing your Microsoft Windows UEFI boot entries. It might be because you have used the Rescapp Fake Microsoft Windows UEFI boot entry option or the Rescapp Hide Microsoft Windows UEFI boot entry option. Or it might be because of your own fault.

Windows has a copy of its own EFI files in its Windows partition. This option let's you reinstall them into your EFI partition so that you can boot into Windows again.

Please note that in some UEFI systems you might need to press an special hotkey (such as F8 or F10) in order to show up the UEFI boot menu in order to be able to choose your added UEFI boot entry.

  • Step 1

    Rescapp searches for EFI partitions on your system.
  • Step 2

    If such partitions have been found, Rescapp displays them. At the same time you are prompted:

    Which EFI System partition?

    You are supposed to choose, your EFI System partition and press the OK button.

  • Step 3

    Rescapp searches for Windows system partitions present, on your sytem.
  • Step 4

    You are prompted:

    Which Windows partition?

    You are supposed to, choose the Windows partition which its UEFI boot entries you want to reinstall.

  • Step 5

    You are optionally informed about the EFI directory backup process.

    You will get (if an error happens):

    [ERROR] Backup of EFI directory went wrong.

  • Step 6

    Rescapp tries to reinstall Microsoft Windows UEFI boot entries.
  • Step 7

    You are informed about the Reinstall Microsoft Windows UEFI boot entries process.

    You either get:

    [SUCCESS] The reinstallation, of the Microsoft Windows UEFI boot entries was OK ! :)

    or

    [ERROR] The reinstallation, of the Microsoft Windows UEFI boot entries was not ok :(
